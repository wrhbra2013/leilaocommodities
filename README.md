# Leilão Commodities

Site estático de leilão de commodities agrícolas.

## Stack

HTML5 + CSS3 + JavaScript Vanilla

## Estrutura

```
├── static/
│   ├── css/style.css       # Estilos
│   └── js/components.js    # Header/footer/nav dinâmico
├── admin/                  # Páginas de administração
│   ├── index.html          # Dashboard
│   ├── leiloes.html        # Gerenciar leilões
│   ├── usuarios.html       # Gerenciar usuários
│   ├── login.html          # Login
│   └── register.html       # Cadastro
├── paginas/                # Páginas públicas
│   ├── comodities.html     # Cotações
│   ├── leilao.html         # Detalhe do leilão
│   ├── leiloes.html        # Lista de leilões
│   └── meus-lances.html    # Lances do usuário
└── index.html              # Landing page
```

## Uso

Abra `index.html` no navegador ou sirva com qualquer servidor HTTP estático.
